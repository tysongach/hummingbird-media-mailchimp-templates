# Hummingbird Media MailChimp Templates

## Setup

1. Get the code:

  ```
  git clone https://github.com/tysongach/hummingbird-media-mailchimp-templates.git
  ```

1. Set up your machine:

  ```
  bin/setup
  ```

1. Run the app:

  ```
  bundle exec middleman
  ```

  ```
  open http://localhost:4567
  ```
